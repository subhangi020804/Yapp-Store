# 🛠️ Yapp Store – Home Services Web Application

Yapp Store is a responsive front-end web application that offers home services such as cleaning, electrical repairs, and maintenance. This project was created as part of a web development project to practice and showcase front-end development skills using modern technologies.

## 📌 Features

- 📋 Categorized service listings (Cleaning, Electrical, Maintenance)
- 📅 Booking form to simulate real-time service requests
- 📞 Contact form for queries and support
- 📱 Fully responsive layout with modern UI
- ♿ Accessible design for all users
- 💻 Cross-device compatibility

## 🧰 Tech Stack

- **HTML5** – Markup structure  
- **CSS3** – Styling and layout  
- **Bootstrap 5** – Responsive design and components  
- **JavaScript** – Interactivity and form handling
